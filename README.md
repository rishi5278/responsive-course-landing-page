# Responsive Course Landing Page 🚀

A modern and fully responsive course landing page built using **HTML5** and **CSS3**.

## 📌 Features

* Responsive Navigation Bar
* Modern Hero Section
* Course Information Card
* Pricing Section
* CTA (Call To Action) Buttons
* Mobile Responsive Design
* Flexbox Layout
* CSS Variables
* Responsive Typography using `clamp()`
* Beautiful "Get Placed" Bloom Text Effect

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* CSS Variables
* Remix Icons

## 📱 Responsive Design

The website is optimized for:

* Desktop
* Tablet
* Mobile Devices

## 🎯 What I Learned

While building this project, I practiced:

* Responsive Web Design
* Flexbox Layouts
* CSS Variables
* Media Queries
* Fixed Navigation Bars
* Modern CSS Techniques
* UI Structuring and Layout Design
